ID V/S Classes

ID

ID is an atrribute a unique identifier, assigned to only one HTML element within a page. It is often used for unique styling and JavaScript manipulations. If you have given an id to an element then you cannot give that id to another element.

Class

The class attribute lets you give the same name to mulitple HTML elements. That way, you can easily change their look or behavior all at once. Classes are not unique and can be assigned to multiple elements. They are generally used for applying the same styles or behaviours to a group of elements.
