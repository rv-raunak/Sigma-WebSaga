What is Flexbox?

It was introduced in CSS 3 as a measure to design the layout of the page so that aligning the content on the page becomes simple (especially putting the content in the centre). So, it kinda the replaced the earlier methods of float or positioning.

Concept of Main ans Cross Axis

Main Axis and Cross Axis are there which change according to the Flex Direction (For Row it would be different and for column it would different etc.)

Points to remember

  • The direction in which the items are laid(directed)is always the main axis
  • The direction in which the items are not laid is always the cross axis
  • Orientation of main and cross axis change according to the flex-direction (Check for the attached screenshots to see it)
  • Remember that the justify-content and align properties work according to the Main and cross axis
    • Example: if we say justify-content: center, so it actually centers it according to the main axis.(same for other positions) (Thus, it doesn't see horizontal or vertical) (in short justify-content centres works according to the main axis)
    • Similarly, the align-items will work accrding to the cross axis
    • Thus, both the properties don't work according horizontal or vertical but they will first determine the main and cross axis and then work accordingly.
  • Other Terminologies:
    • Main start: Origin of main axis
    • Main end: End of main axis
    • Main Size: Length of main axis i.e. Distance from main start to main end
    • same definations can be extended for Cross start, cross end and cross size
