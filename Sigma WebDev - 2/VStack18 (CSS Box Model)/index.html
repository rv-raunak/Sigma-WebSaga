Box Model Concept

Everything inside html/css (be it all elements of html like container, div, span etc.) is nothing but a box which has three properties(content->padding->border->margin)(The latter ones contain the previous ones respectively)

(The total height and width is equal to the sum of content + padding + margin)(Total height and width means the total displaying space consumed by the element)

Know More About Box Model



    Formlua to calculate the total height and width or the total space consumed by the element with padding and margin

  • Total Width = Width + Left Padding + Right Padding + Left Border + Right Border + Left Margin + Right Margin
  • Total Height = Height + Top Padding + Bottom Padding + Top Border + Bottom Border + Top Margin + Bottom Margin


Concept Of Margin-Collapse

This states if the margins of the two objects(boxes) is overlapping then the maximum margin among both will be selected and that will be applied to between them only once
