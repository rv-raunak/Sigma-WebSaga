  1. Inline CSS: Can stylize individual elements but not suitable for bulk edit.(Try to avoid it)
  2. Internal CSS: can style all the elements in a page at once but the html file length grows overtime leading to complexitiy in managing the code.(Use it when te need for css stylizing is less)
  3. External CSS: Same as Internal CSS but we write our code in an extrenal file for easier code management. (Usually used for a production-rate website as the sheet is editable as well as detachable)
